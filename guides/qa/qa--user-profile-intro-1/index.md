## Module Requirements
### **Functional Requirements**
  - Check that users can update personal details like name or contact info without changing login-related information.
  - Confirm that profile fields show the latest saved information and update correctly.
  - Ensure updates apply only to the currently logged-in user.
  - Verify that update screens show clear success and error messages.
### **Security Requirements**
  - Confirm that only signed-in users can access the profile update features.
  - Check that users cannot update another userâ€™s profile.
  - Test that all input fields block incorrect or harmful data.
  - Ensure profile changes happen through secure, verified steps.
### **Performance Requirements**
  - Measure how quickly profile information loads and updates.
  - Check that the system can handle many users updating their profiles at the same time.
  - Review if loading non-sensitive profile data uses performance-friendly techniques.
  - Confirm profile data stays consistent when reloading or switching pages.
### **Accessibility & Responsiveness**
  - Check that all input fields and buttons are easy to access with keyboard and screen readers.
  - Test layout and features on both desktop and mobile devices.
  - Confirm that profile update features adjust well to different screen sizes.
  - Ensure users receive visible feedback after saving or entering incorrect data.
