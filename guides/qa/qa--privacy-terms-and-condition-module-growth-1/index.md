### **Functional**
  - Check if the system shows new versions of the policy when content changes.
  - Confirm users are asked to re-accept when there's an update.
  - Make sure each new version has a clear label (like a date).
  - Confirm the system remembers which version each user agreed to and when.
  - Make sure there's a place where users can see older versions if needed.
